Fiddler Calls Visualizer

Project Overview

Built a web-based tool to visualize HTTP traffic logs captured by Fiddler. Users can upload `.har` files and inspect request/response dependencies through an interactive force-directed graph. This helps developers and testers analyze API behavior and session flows more intuitively.

Key Features

  • Visualizes HTTP traffic using a force-directed graph layout
  • Parses `.har` files to extract session and call data
  • UI panels for viewing headers, cookies, body, and query params
  • Filtering and grouping calls based on domain and session ID
  • Search functionality for quick inspection of specific requests
  • Designed for performance with efficient rendering and interaction

Technologies Used

Node.js Express.js D3.js Bootstrap HTML5
